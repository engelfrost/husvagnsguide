# Snabbstart

## När du kommer fram

- [ ] Veva ner stödbenen/fötterna.
- [ ] Koppla in el om elplats finns.
- [ ] Sätt på gasen när vagnen står stilla och ska användas.
- [ ] Öppna gasspjällen.
- [ ] Sätt på kylen.
- [ ] Koppla gråvattentanken.
- [ ] Fyll på vatten.
- [ ] Öppna takluckorna i ventilationsläge när ni bor i vagnen.

## Innan ni kör

- [ ] Stäng gasen.
- [ ] Stäng och haspa takluckor och fönster.
- [ ] Lyft ner tunga saker ur överskåpen.
- [ ] Töm vattentank och gråvatten.
- [ ] Veva upp stödbenen/fötterna.
- [ ] Ta bort lösa saker från bänk och bord.
- [ ] Kontrollera att släpets belysning fungerar.

Mer detaljer finns i [Ställa upp](uppstallning.md) och [Inför avfärd](avfard.md).
