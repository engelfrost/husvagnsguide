# Lasta husvagnen

Det är mycket viktigt att lasta ett släp rätt.

## Grundregel

- Minst 60 % av vikten ska ligga mot fören.
- Så mycket av tyngden som möjligt ska ligga lågt och över hjulaxeln.
- Undvik tung last längst bak i husvagnen.
- Lyft ner tunga saker ur överskåpen före körning.

Fel viktfördelning kan ge dåliga och farliga köregenskaper.
