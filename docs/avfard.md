# Inför avfärd

Gå igenom checklistan innan husvagnen flyttas eller lämnas tillbaka.

## Invändigt

- [ ] Stäng takluckor.
- [ ] Haspa fönster fram.
- [ ] Haspa fönster bak.
- [ ] Haspa fönstret till toaletten.
- [ ] Dra upp rullgardin fram och bak så att sikten inte blockeras.
- [ ] Lyft ner tunga saker ur överskåpen.
- [ ] Kontrollera skåpsluckor.
- [ ] Lås garderoben.
- [ ] Spärra kylskåpsdörren med packning.
- [ ] Ta ner bordslampan.
- [ ] Haspa skjutdörren.
- [ ] Stäng toalettdörren.
- [ ] Ta bort lösa saker från bänk och bord.

## Utvändigt och system

- [ ] Stäng av gasen.
- [ ] Töm vattentanken.
- [ ] Koppla ur gråvatten.
- [ ] Veva upp stödbenen/fötterna.
- [ ] Lås dörren.
- [ ] Lås luckan.

!!! danger "Kontrollera innan körning"
    Kontrollera att takluckor, fönster, gas och stödben verkligen är säkrade innan ni kör.
