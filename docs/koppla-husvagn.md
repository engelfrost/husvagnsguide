# Koppla husvagnen

Följ stegen lugnt och kontrollera kopplingen innan körning.

1. Fäst säkerhetsvajern i bilen med karbinhaken.
2. Placera kulhandsken över dragkroken.
3. Sänk ner kulhandsken över dragkroken.
4. Säkerställ att handtaget på kulkopplingen har fällts ner och att markören ändras.
5. Veva ner stödhjulet lätt för att kontrollera att släpet sitter fast ordentligt på bilen.
6. Veva upp stödhjulet så att hjulet inte kan rotera.
7. Lossa klämfästet, dra upp stödhjulet och justera klämfästet.
8. Anslut släpets elkontakt.
9. Kontrollera att belysningen fungerar.

!!! warning "Viktig kontroll"
    Om kulkopplingen inte visar rätt markering, eller om husvagnen inte sitter fast när du testar med stödhjulet: kör inte vidare förrän kopplingen är rätt.
