# Ställa upp husvagnen

Gå igenom checklistan när husvagnen står på plats.

- [ ] Veva ner stödbenen/fötterna.
- [ ] Sätt på gasen.
- [ ] Öppna gasspjällen.
- [ ] Öppna takluckorna i ventilationsläge.
- [ ] Koppla in elen om elplats finns.
- [ ] Sätt på kylen.
- [ ] Koppla gråvattentanken.
- [ ] Fyll på vatten.

!!! warning "Under färd"
    Gasen ska vara avstängd och takluckorna ska vara helt stängda under körning.
